# Dosato Language syntax highlighting

## How to use

1. Install the extension
2. Open a Dosato file (default extension is `.to`)

## Features

- Syntax highlighting
    Syntax highlighting for Dosato language
    Variables, functions, comments, strings, numbers, etc.
